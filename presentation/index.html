Handling events is hard

Handling data is easy easier

Here's how we handle data


const characters = [
  { name: 'Philipp', age: 25 },
  { name: 'Leela', age: 26 },
  { name: 'Hubert', age: 163 },
  { name: 'Amy', age: 22 }
]

characters.map(character => character.age) // [25, 26, 163, 22]
          .filter(age => age < 100) // [25, 26, 22]
          .reduce((sum, age) => sum + age, 0) // 73

Treat events like data


import Rx from 'rxjs/Rx'

const mouseClicks$ = Rx.Observable.fromEvent(
  document.getElementsByTagName('body')[0],
  'click'
) // [MouseEvent, MouseEvent, MouseEvent, MouseEvent, ...]

mouseClicks$.map(event => event.x) // [6, 7, 923, 200, ...]
            .filter(x => x < 500) // [6, 7, 200, ...]
            .scan((sum, x) => sum + x, 0) // (6 + 7 + 200 + ...)
            .subscribe()
